Using Maven
-----------
First add repository

	<repository>
		<id>tarantool-snapshots</id>
		<url>http://dgreenru.github.com/tarantool-repo/snapshots</url>
	</repository>
and dependency

	<dependency>
		<groupId>org.tarantool</groupId>
		<artifactId>connector</artifactId>
		<version>0.0.1-SNAPSHOT</version>
	</dependency>
